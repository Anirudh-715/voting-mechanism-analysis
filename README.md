# Computational Analysis of Voting Mechanisms

This project simulates different voting systems from social choice theory.

## Algorithms Implemented
- Plurality Voting
- Borda Count
- Condorcet Method
- Condorcet Paradox Detection

## Experiment

The program simulates 100 elections with 50 voters and 4 candidates.  
Random voter preferences are generated to analyze how different voting rules produce different outcomes.

## Technologies Used
Python  
Matplotlib  

## Goal

To analyze how voting rules differ and detect situations where no clear winner exists (Condorcet paradox).
